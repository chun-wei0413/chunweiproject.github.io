拉麵之旅






來到輔大讀書後有搭捷運到處品嚐美食嗎？
在新莊卻發現附近的東西好像都吃膩了嗎？



如果是喜歡吃日本拉麵的人，那請繼續看下去吧！

大家好，我是資工系二年級的李俊威！

我喜歡到處探索美食，所以這次將目標鎖定在很多人喜歡的「拉麵」！ 我會介紹台北地區、捷運都可以到的拉麵店，「道地」、「高cp值」、「獨特」，各式各樣不同風格的拉麵在我的網站上都可以看到哦！ 如果想要更了解拉麵的歷史或是特別之處，也可以在我的選單中看到！

在寒冷的冬天裡能來碗拉麵是再快樂不過的事了，但同時也要吃到好吃的拉麵才會更加快樂。

在這裡，我介紹了12家拉麵店，分別是
1. 豚人拉麵（忠孝復興站）
2.隱家拉麵（中山站）
3.你回來啦拉麵（國父紀念館站）
4.大和家（忠孝復興站）
5.Nagi 凪拉麵（西門站）
6.一蘭拉麵（象山站）
7.半熟堂（中山站）
8.morumoru拉麵（忠孝復興站）
9.鷹流拉麵（國父紀念館站）
10.Due Italian拉麵（市政府站）
11.琉球拉麵（台北車站）
12.花月嵐拉麵（西門站）



最後希望大家會喜歡我的作品，謝謝各位！
